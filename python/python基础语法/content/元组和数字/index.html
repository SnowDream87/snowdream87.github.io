列表的相关操作

  • 1.列表的数学运算：+，*
    ​ 列表1 + 列表2 - 将两个列表中的元素合并参数一个新的列表；（不会修改原列表）
    ​ 列表 * N - N是整数；列表中的元素重复N次产生一个新的列表
  • 2.列表的比较运算：==，!=，大小的比较意义不大
    ​ 列表1 == 列表2 、 列表1 != 列表2
    ​ 两个列表比较大小，不是比较列表的长度，而是比较元素的大小、
  • 3.补充：is的用法
    ​ == -> 判断两个数据的值是否相等
    ​ is -> 判断两个数据的地址是否一样
  • 4.in和not in
    ​ 元素 in 列表 - 判断列表中是否存在指定的元素
    ​ 元素 not in 列表 - 判断列表中是否不存在指定的元素

  • 5.内置函数max(序列), min(序列), sum(序列), len(序列)，list(序列)
    ​ max和min要求系列中的元素类型必须一致：并且元素支持比较运算符
    ​ sum要求序列中的元素必须是数字
    ​ 迭代器和生成器不能用len
    ​ list(序列) - 只有容器型数据才能转换成列表；将序列中的元素做为列表的元素产生一个新的列表

  • 2.列表.extend(序列) - 将序列中的元素添加到列表的最后，没有返回值

  • 3.列表.index(元素) - 获取元素在列表中的正下标(如元素有多个只取第一个,元素必须存在)

    print(nums.index(3))        # 1、

  • 7.copy()
    ​ 列表.copy() - 和列表[:]的效果一样，拷贝列表中的元素产生一个新的列表。这个拷贝是浅拷贝

    nums = [1, 2, 3]

    元组

    • 1.什么是元组(tuple)
      元组就是不可变的列表
      ​ 元组是容器型数据类型(序列)，将小括号做为容器的标志，多个元素用括号隔开(元素1，元素2，…)
      ​ 特点：不可变(不支持增删改)，有序(支持下标操作)
      ​ 元组中的元素 - 和列表要求一样

    • 2.元组的表示
      ​ 单个元素的元组(元素，)​（），（10），（10, 20）

    • 3.获取元组的元素
      ​ 列表中获取元素的范式元组都支持;注意：切片的结果是元组
      ```python #语法：多个变量某一个变量前带 = 元组 #让不带*的变量去元组中获取元素，剩下的全部给带*的变量(带*的变量会变成列表) student = (‘小米’, 28, ‘std001’, 20, 25, 35) name, age, study_id, *scores = student print(name, age, study_id, scores)

    x, *y, z = student print(x, y, z) # [小米 [28, ‘std001’, 20, 25] 35 ```

数字

python中数字相关的类型有：int，float，bool，complex(复数)
+ 1.整型
​ 所有的整数对应的数字类型就是整型
​ int(数据) -> 将数据转换成整型。所有的小数、布尔值、和部分字符串可以砖汉成整数字符串中只有去掉 引号后本身就是一个整数的字符串才能转换成

  print(int(3.22))    # 3
  print(int(True), int(False))     # 1 0
  print(int(45), int('-123'))      # 45 123
  print(int())    # 0

  • 2.浮点型
    ​ 所有小数对应的类型就是浮点型，浮点型对应的数据类型只有float。支持科学计数法：3e4
    ​ float(数据) - 整型、布尔、和部分字符串可以转换成浮点型,字符串中只有去掉引号后本身就是一个数字的才能转换成浮点型

    print(3e4)  # 30000.0
print(float(100), float(True))  # 100.0 1.0
    • 3.布尔
      ​ 布尔中的True本质就是整数1，False本质就是整数0
      ​ bool(数据) - 所有的数据都能转换成bool，所有为0为空的值都会转换成False，其他都是True
    • 4.复数
      ​ 由实部和虚部组成的数字胶复数：a+bj，对应的类型是complex,并直接支持复数运算
      python a = 10 + 20j print(a, type(a))
