字典

什么是字典（dict）

  • a.字典是容器型数据类型(序列)，将大括号{ }做为容器的标志，里面多个元素通逗号给开
        特点：可变(支持增删改)，无序(不支持下标操作)
  • b.字典中的元素
         ，以’键:值’的形式成对出现 - {键1:值1，键2:值2，….}
        字典存储数据主要为了存值，键只是为了区分不同的值而存在的
  • c.要求：
        键值对中的键 - 键必须不可变；唯一的
        键值对的值 - 和列表元素一样，任何数据都可以作为值
    ###字典的增删改查
    #####1.查 - 获取字典的值

1）. 获取单个元素的值
    字典[key] - 获取字典中指定key对应的值；如果key不存在，会报错
    字典.get(key) / 字典.get(key，默认值) - 获取字典中指定key对应的值；如果key不存在返回值为None，或默认值

2）.遍历
    for - in 字典直接拿到的是key

dog = {'name': '毛孩子', 'age': '2'}
print(dog.get('name'))

# 常用遍历
print('=============1============')
for key in dog:
    print(key, dog[key])

# 以下不能用，能看懂就行
print('=============2============')
for value in dog.values():
    print(value)

print('=============3============')
for key, value in dog.items():
    print(key, value)
2.增/改 - 增加字典的值

    增 - 添加键对；改 - 修改字典中某个key对应的值
    语法：字典[key] = 值 - 当key不存在就在字典中添加键值对’key’:值 ；当key存在的时候就修改对应的值

dog['color'] = '黄色'
print(dog)
dog['color'] = '白色'
print(dog)
3.删 - 删除key对应的值

    1）del 字典[key] - 删除指定key对应的键值对
    2）字典.pop[key] - 取出字典中key对应的值,返回被取出来的值

相关运算

    字典不支持数学运算符，比较运算符，只支持 ==， !=
###相关操作：in / not in ，max，min，sum，len，dict
    key in 字典 - 判断字典中是否存在某个键
    max(字典) - 求的是字典中key的最大值
    max，min，sum，len - 针对的是key的操作
    dict(数据) - 将指定数据转换成列表
         数据的要求：
             a.数据本身是一个序列
             b.序列中的元素，必须有且只有两个元素的小序列

print(dict([(1, 2), ['小狗', 3], 'qw']))
dic = dict([(1, 2), ['小狗', 3], 'qw'])
print(dic)

相关方法

    1)dict.fromkeys(序列， 值1 = None) - 创建一个新字典，将序列中的元素做为字典的key，将值1做为每个key对应的value

new_dict = dict.fromkeys(['a', 'b', 'c', 'd'], 10)
print(new_dict)

    2)字典.item() - 将字典中的键值对转换成元组作为容器中的元素
    3)字典.value()，字典.key()
    4).字典.setdefault(key, vlaue = None) - 在字典中添加键值对( )
    5)字典1.update(字典2) - 将字典2中的键值对全部添加到字典1中

dic.update(new_dict)
print(dic)

#集合

什么是集合(set)

1）集合
    集合是容器型数据类型；将{}作为容器的标志，多个元素用都还隔开(和字典不一样，集合的元素是独立的数据不是键值对)
    特点：
        可变(增删)、无序(不支持下标操作)
        自带去重功能
    注意：
        {}表示的是空子典；
        set()创建空集合

2）元素 - 不可变的数据；同一个元素只能有一个(唯一性)

集合元素的增删改查

1)查 - 只支持遍历，不能单独获取具体某一个元素
2)增:
    集合.add(元素) - 可以在集合中添加一个元素
    集合.update(序列 ) - 将序列中的所有元素添加到集合中
3)删：
     集合.remove(元素) - 删除集合中指定的元素，元素必须存在
4)改(集合不支持修改元素的操作)

数学集合运算：

1）求并集： 集合1 | 集合2 —— 将两个集合的元素合并，并去重
2）求交集： 集合1 & 集合2 —— 获取两个元素的公共部分
3）求差集： 集合1 - 集合2 —— 获取集合1中除去集合2的元素剩下的部分
4）对称差集：集合1 ^ 集合2 —— 两个集合除了公共部分剩下的部分
5）包含： 集合1 > 集合2 —— 判断集合1是否包含集合2
