列表

1.什么是列表(list)
       列表是容器型数据类型(序列),将[]做为容器的标志，多个元素用逗号刚隔开。
       特点:可变的(指的列表中元素可变，元素支持增删改)、有序的(支持下标操作)

2.声明列表
       变量名 = [元素1，元素2，元素3，…..]
       [] -> 空列表

3.列表中的元素：
       任何类型的数据都可以作为列表元素：同一个列表中的数据类型可以不一样

4.获取类表中的元素
       1).获取单个元素
       a.语法：
              列表[下标] - 获取列表中指定下标对应的元素
       b.说明：
              列表 - 结果是列表的表达式(列表数据、列表变量)
       [] - 固定写法
              下标 - 下标又叫索引；列表中每个元素都有固定的下标值来表示元素在列表的位置
       范围1：0,1,2,3， 长度-1(从前往后)
       范围2：-1，-2，-3，-长度(从后往前)
       注意：下标不能越界
2).获取部分元素(切片)
       列表切片的结果是列表
       a.语法：
             列表[开始下标:结束下标:步长] - 从开始下标开始，每次增加步长，去取下一个元素直到取到结束下标为止
       注意：步长为正(负)表示开始下标到结束下标从前往后取(从后往前)，所以开始下标对应的元素，
       必须在结束下标对应元素的前面(后面)，否则为[]
       省略下标： 可省略开始坐标，结束坐标，步长
       省略开始下标：步长为正，从第一个开始往后取；步长为负，从最后一个元素往前取
       省略结束下标：步长为正，取到最后一个为止；步长为负，取到第一个为止
       两个都省：全部取出，步长为取的方向
3).遍历 - 将列表中的元素一个一个的取出来
       a.直接遍历元素
              for 变量 in 列表
                     循环体
       b.通过遍历下标遍历列表元素
      len() - 用来获取列表元素的个数
      for index in range(len(列表))

sores = [1, 2, 3, 4, 5]
for sore in sores:
    print(sore)

for index in range(len(sores)):
    print(index, sores[index])

增删改

1.增(增加列表中的元素)
      1)列表.append(元素) – 在列表的最后添加一个元素
      2)列表.insert(下标， 元素) – 在列表指定下标前插入一个元素

films = ['复仇者联盟', '钢铁侠', '哈利波特', '肖胜克的救赎', '闪灵']
films.append('咒怨')
films.insert(3, '风之谷')
print(films)

2.删(删除列表中的元素)
      1)del 列表[下标] – 删除列表中指定下标对应的元素（下标不能越界）
      2)列表.remove(元素) – 删除列表中第一个指定元素（元素必须存在）
      3)列表.pop(下标) – 取出列表中指定下标的元素（返回被取出来的元素）
             列表.pop() – 取出列表中最后一个元素（返回被取出来的元素）

del films[-1]
print(films)

3.改（修改列表中的元素值，且下标不能越界）
      列表[下标] = 新值

#习：删除分数列表scores中分数低于60分的成绩
scores = [78, 56, 40, 66, 70, 12, 45, 59, 90]
lists = []
for score in range(len(scores)):
    if scores[score] >= 60:
        lists.append(scores[score])
print(lists)
