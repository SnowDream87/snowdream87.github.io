Python PIL

Jul 21, 2019
1 min read

  PIL是Python第三方图像处理库，该功能强大可以对图像进行很多处理，比如图像剪切、粘贴、滤镜、缩略图、镜像、图片呈现方式、旋转等功能。
  在使用PIL是我们也要与Image，Imagefilter这两个库配合使用。以下就是一些PIL的简单教程实例。

#我先创建一个空白图片出来，方便我将处理后的图片粘贴在上面  
#Image.new(模式，大小，颜色)  
whileImage = Image.new('RGB', (1366, 768), (255, 255, 255)) 

#加载图像，图片对象 = Image.open（图片路径）  
image = Image.open('image/0201.jpg')  

#再获取到图片的宽和搞  
width, height = image.size  

#用thumbnail（）方法将图像像素改为以下值  
image.thumbnail((455, 256))  

#将图片对象用filter方法进行滤镜处理。同时再新建一个图片对象  
#并且将滤镜处理后的图片放入该对像，以下是几种滤镜效果  
#ImageFilter.EMBOSS   -  浮雕效果  
#ImageFilter.FIND_EDGES - 泼墨效果  
#ImageFilter.SHARPEN  -  锐化滤波  
#ImageFilter.SMOOTH  -  平滑滤波  
#ImageFilter.EDGE_ENHANCE_MORE - 边界增强滤波（程度更深）  
#ImageFilter.EDGE_ENHANCE - 边界增强滤波  
#ImageFilter.DETAIL - 细节滤波  
#ImageFilter.CONTOUR - 轮廓滤波(铅笔画)  
#ImageFilter.BLUR - 模糊滤波  
image_1 = image.filter(ImageFilter.EMBOSS)  
image_2 = image.filter(ImageFilter.FIND_EDGES)  
image_3 = image.filter(ImageFilter.SHARPEN)  
image_4 = image.filter(ImageFilter.SMOOTH)  
image_5 = image.filter(ImageFilter.EDGE_ENHANCE_MORE)  
image_6 = image.filter(ImageFilter.EDGE_ENHANCE)  
image_7 = image.filter(ImageFilter.DETAIL)  
image_8 = image.filter(ImageFilter.CONTOUR)  
image_9 = image.filter(ImageFilter.BLUR)  

#用paste方法将各种滤镜效果粘贴在我最初新建的空白图片里  
#图片对象.paste(图片对象2,(x坐标，y坐标))  
#注意：因为我宽度除3是小数因此需要强制转换成int类型  
whileImage.paste(image_1, (int(width/3*0), 0))  
whileImage.paste(image_2, (int(width/3*1), 0))  
whileImage.paste(image_3, (int(width/3*2), 0))  
whileImage.paste(image_4, (int(width/3*0), int(height/3)))  
whileImage.paste(image_5, (int(width/3*1), int(height/3)))  
whileImage.paste(image_6, (int(width/3*2), int(height/3)))  
whileImage.paste(image_7, (int(width/3*0), int(height/3*2)))  
whileImage.paste(image_8, (int(width/3*1), int(height/3*2)))  
whileImage.paste(image_9, (int(width/3*2), int(height/3*2)))  
whileImage.show()  


#rotate（）方法可以将图像逆时针角度旋转  
whileImage=whileImage.rotate(45) 
whileImage.show()  


#convert（）方法，有两种模式，"L"将图像以灰度图呈现,"RGB"以彩色图呈现  
whileImage=whileImage.convert("L")  
whileImage.show()  


#crop方法用于剪切&emsp;图片对象.crop((起点x坐标 ，起点y坐标, 终点x坐标， 终点y坐标))  
image=image.crop((0, 0, 600, 600))  
image.show()  


image1 = Image.open('image/4011.jpg')  
#原图像  
image1.show() 


#左右镜像的方法，transpose(Image.FLIP_LEFT_RIGHT)  
image1 = image1.transpose(Image.FLIP_LEFT_RIGHT)  
image1.show()  


#上下镜像的，transpose(Image.FLIP_TOP_BOTTOM)
image1 = image1.transpose(Image.FLIP_TOP_BOTTOM)  
image1.show()  


#色数分离，分离r&emsp;,g&emsp;,b&emsp;红绿蓝三原色    
r, g, b = image.split()  
r.show()  
g.show() 
b.show()



以上就是PIL库的一些简单实用的案例
